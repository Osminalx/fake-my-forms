data-* attribute hints

No label/name/autocomplete — field type detected via data-testid, data-cy, etc.

data-cy="phone-field"
data-test="firstName"
data-field="address"
data-name="zipCode"
data-testid="password-field"
data-cy="company-input"

Priority conflict scenarios

autocomplete beats label; label beats name/id; data-* beats name/id.

signal priority
label="Username" but autocomplete="email" → email wins
label="Email address", name="username" → label wins
data-testid="phone-field", name="email_addr" → data-* wins
